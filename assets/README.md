# MarketMove App - Assets
Este directorio contiene los recursos gráficos de la aplicación.

## Estructura
```
assets/
├── images/           # Imágenes generales (logos, fondos, etc.)
│   └── logo_placeholder.svg
└── icons/            # Iconos personalizados
```

## Notas
- El logo placeholder debe ser reemplazado por el logo final del cliente
- Los iconos deben seguir las guías de Material Design
- Formatos recomendados: PNG (con múltiples resoluciones) o SVG
